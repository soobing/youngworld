Windows에 Claude Code 설치하기

복잡한 WSL 설치 없이, PowerShell 한 줄로 가장 쉽게 설치하는 방법이에요.

✅ WSL도, Node.js도 필요 없어요. 아래 Native 설치 방법은 Windows에서 그대로 동작합니다.

0시작 전에

  • 운영체제: Windows 10 (1809 이상) 또는 Windows 11
  • 계정: Claude Pro·Max·Team·Enterprise 중 하나가 필요해요. 무료 플랜은 Claude Code를 쓸 수 없어요.
  • 인터넷 연결

1가장 쉬운 설치 추천

PowerShell을 관리자 권한 없이 열고, 아래 한 줄을 붙여넣고 Enter 하면 끝이에요.

① PowerShell 열기

시작 메뉴에서 "PowerShell"을 검색해 실행해요. (관리자 권한 필요 없음)
창 왼쪽에 PS C:\Users\...> 처럼 PS가 보이면 PowerShell이 맞아요.

② 설치 명령 실행

PS> irm https://claude.ai/install.ps1 | iex

잠시 기다리면 자동으로 설치돼요. (이후 업데이트도 백그라운드에서 자동으로 됩니다.)

💡 더 간단한 대안 — WinGet. Windows에 기본 내장된 설치 도구예요. 아무 터미널에서: 
> winget install Anthropic.ClaudeCode
(WinGet은 자동 업데이트가 안 돼서, 가끔 winget upgrade Anthropic.ClaudeCode 로 올려줘야 해요.)

2설치 확인

새 터미널을 열고(중요: 방금 창 말고 새로 열기) 아래를 실행해요.

PS> claude --version
# 2.1.211 (Claude Code) 처럼 버전이 나오면 성공 ✅

더 자세히 점검하려면:

PS> claude doctor   # 설치·설정 상태를 진단해줘요

3실행 & 로그인

  1. 작업할 폴더에서 터미널을 열고 claude 를 실행해요.
  2. 브라우저가 열리면 Claude 계정으로 로그인하고 승인해요.
  3. 터미널로 돌아오면 대화가 시작돼요. 🎉

4(선택) Git for Windows 설치하면 더 좋아요

필수는 아니에요. 하지만 Git for Windows ↗를 깔면 Claude Code가 Git Bash로 명령을 실행해 더 매끄러워요. (없으면 PowerShell로 동작해요.)

혹시 Claude Code가 Git Bash를 못 찾으면, %USERPROFILE%\.claude\settings.json 에 아래를 넣어줘요:

{
  "env": {
    "CLAUDE_CODE_GIT_BASH_PATH": "C:\\Program Files\\Git\\bin\\bash.exe"
  }
}

5자주 겪는 문제 & 해결 🛠️

이런 메시지가 뜨면이렇게 해결해요
'irm' is not recognized 지금 CMD 창이에요(PowerShell 아님). 시작 메뉴에서 PowerShell을 열어 다시 실행하세요. (PS 표시 확인)
'claude' is not recognized / command not found 설치는 됐지만 경로(PATH)가 아직 반영 안 된 거예요. 터미널을 완전히 닫고 새로 여세요. 그래도 안 되면 PowerShell에서: 
[Environment]::SetEnvironmentVariable('PATH',
  "$env:PATH;$env:USERPROFILE\.local\bin", 'User')
실행 후 새 터미널에서 claude --version 다시 확인.
설치는 됐는데 옛 버전이 뜸 %USERPROFILE%\.local\bin 이 PATH 맨 앞에 있는지 확인하고 새 터미널을 여세요.
로그인이 안 됨 / 권한 없음 무료 플랜은 Claude Code를 쓸 수 없어요. Pro·Max 등 유료 플랜 계정으로 로그인하세요.
회사 네트워크에서 403·연결 실패 사내 방화벽/프록시 때문이에요. 프록시 주소를 넣고 다시 설치: 
PS> $env:HTTPS_PROXY = 'http://proxy.회사주소:8080'
PS> irm https://claude.ai/install.ps1 | iex
터미널이 무섭거나 계속 실패 터미널 없이 쓰는 Claude Code 데스크톱 앱도 있어요 — claude.com/download 에서 Windows용을 받으면 클릭만으로 설치·사용할 수 있어요.
