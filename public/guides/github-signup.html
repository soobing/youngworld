🐙 GitHub

GitHub 가입하는 법

GitHub은 코드를 저장하고 친구들과 공유하는 개발자들의 공간이에요.

🎯
이번 미션은 「GitHub에 가입하고, 내 Username을 쪽지로 공유하기」 예요.
아래를 천천히 따라 하면 누구나 5분이면 가입할 수 있어요!

🎒 준비물

  • 인터넷이 되는 컴퓨터나 휴대폰
  • 내가 확인할 수 있는 이메일 주소 1개

🚀 가입 순서

  1. github.com 에 접속해요

    주소창에 github.com 을 입력하고 이동해요.

    📸 이런 화면이 나와요

    🔒 github.com

  2. 오른쪽 위 Sign up 버튼을 눌러요

    “Sign up(가입)” 은 새로 가입, “Sign in(로그인)” 은 이미 있는 계정으로 들어가기 예요. 우리는 처음이니 Sign up!

    📸 화면 맨 위(헤더)

    🐙 GitHub Sign in Sign up ←

  3. 이메일 주소를 입력해요

    평소 쓰는 이메일을 적고 Continue(계속) 를 눌러요.

    📸 이메일 입력 칸

    Enter your email

    yeongwol.soo@example.com
    Continue

  4. 비밀번호를 만들어요

    8자 이상, 남들이 못 맞힐 만한 걸로 정해요. 영어·숫자를 섞으면 더 안전해요.

    📸 비밀번호 입력 칸

    Create a password

    • • • • • • • • • •
    Continue
    💡 정한 비밀번호는 안전한 곳에 적어 두세요. 나중에 로그인할 때 필요해요.

  5. 사용자 이름(Username)을 정해요 가장 중요! ⭐

    Username은 전 세계에서 딱 하나뿐인 나의 아이디 예요. 미션에서 쪽지로 공유할 이름이 바로 이거예요.
    • 영어 소문자·숫자·하이픈(-) 만 쓸 수 있어요. (한글·띄어쓰기 ✕)
    • 이미 누가 쓰는 이름이면 못 써요 → 숫자나 단어를 붙여서 바꿔요.

    📸 사용 가능한 이름이면 초록 체크가 떠요

    Enter a username

    yeongwol-soo ✓

    ✓ Username yeongwol-soo is available.

    📸 이미 있는 이름이면 빨간 글씨가 떠요 → 다른 이름으로!

    Enter a username

    soo

    ✗ Username soo is not available.

    💡 좋은 이름 예시: yeongwol-soo, minji2011, coding-hana
    Username은 나중에 바꾸기 번거로워요. 오래 쓸 이름으로 신중히 골라요.

  6. 내가 사람인지 확인해요

    간단한 퍼즐(그림 맞추기 등)이 나오면 안내대로 풀어요. 로봇이 아니라는 걸 확인하는 단계예요.

  7. 이메일로 온 인증 코드를 입력해요

    가입할 때 적은 이메일함을 열면 GitHub이 보낸 숫자 코드가 있어요. 그 숫자를 입력하면 끝!

    📸 인증 코드 입력 칸

    Enter code

    8 3 0 5 1 9

  8. 🎉 가입 완료!

    이제 GitHub 회원이 되었어요. 화면이 내 대시보드(첫 화면) 로 바뀌어요.

🔎 내 Username은 어디서 확인해요?

가입할 때 정했지만 헷갈릴 수 있어요. 두 가지 방법으로 확인할 수 있어요.

  1. 오른쪽 위 동그란 프로필 사진을 눌러요

    메뉴 맨 위에 “Signed in as ○○○” 라고 나오는데, 그 ○○○ 이 내 Username 이에요.

    📸 프로필 메뉴

    Y← 이 사진을 클릭
    Signed in as
    yeongwol-soo

  2. 주소창의 내 프로필 주소로도 알 수 있어요

    github.com/내이름 이 내 프로필 페이지예요. 슬래시(/) 뒤가 Username 이에요.

    📸 주소창

    🔒 github.com/yeongwol-soo
⚠️ Username ≠ 이메일, ≠ 비밀번호 예요. 미션에서 공유하는 건 Username(예: yeongwol-soo) 하나예요. 비밀번호는 절대 남에게 알려주면 안 돼요!

✅ 미션 완료하기

확인한 Username 을 영월드에서 쪽지로 보내면 미션 끝!

  1. 영월드 오른쪽 위 ⚙️ 내 설정 을 눌러요.
  2. 쪽지 탭에서 받는 사람을 'soobing' 으로 골라요.
  3. 내용에 내 GitHub Username 을 적어 보내요. (예: 제 깃허브 아이디는 yeongwol-soo 예요!)
🙋
혼자 하다가 모르는 게 있으면, soobing 에게 카카오톡으로 언제든지 물어봐요! 😊
