GitHub 가입하는 법

코드를 저장하고 친구들과 공유하는 개발자들의 공간이에요.

  1. github.com 에 접속해요. 주소창에 https://github.com 을 입력합니다.
  2. 오른쪽 위 Sign up(가입) 버튼을 눌러요.
  3. 이메일 주소를 입력해요. 평소 쓰는 이메일이면 돼요.
  4. 비밀번호를 정해요. 남들이 모를 만한 걸로 만들어요.
  5. 사용자 이름(Username)을 정해요. 전 세계에서 하나뿐인 이름이라, 원하는 게 이미 있으면 숫자를 붙여봐요. (예: yeongwol_soo)
  6. 이메일로 온 인증 코드를 입력하면 가입 완료!
💡 학교/부모님 이메일로 가입해도 괜찮아요. 사용자 이름은 나중에 바꿀 수 있지만, 되도록 오래 쓸 이름으로 정해요.
