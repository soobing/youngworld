# 영월드 맵 구상

## 개요
이 강의를 진행하면서 youngworld 라는 게임맵을 만들어서, 강의자료 및 실습 결과물을 모두 해당 맵에 올리도록 진행 예정

## 컨셉
닌텐도 동물의 숲과 같은 컨셉으로, 수업에 참여하는 5명 학생과 선생님인 나까지 총 6명이 존재하는 섬.
학생들이 수업을 들으면서 이 섬을 더 발전 시키도록 개발 할 예정이고 나는 대략적인 틀만 먼저 제공

- 브라우저로 접속 가능한 웹 게임 
- 초기 섬에는 5명의 학생, 1명의 선생님 아바타, 1명의 게스트 아바타 = 총 7명의 아바타 존재
  - 아바타 꾸미기는 각자 AI를 활용하여 코딩해서 진행할 예정이므로 최대한 간단하게 제공
  - 1명의 게스트 아바타는 그냥 뷰어 역할만 함. 다른 아바타들 처럼 무언가 컨텐츠를 생성하고 대화를 나누고 할 수 없음. 자세한 기능 명세는 개발하면서 더 고도화 할 예정이나, 애초에 권한 분리가 필요함.
  - 1명의 선생님은 admin 권한을 가짐. 
    - 학생 아바타 or 게스트 아바타를 추가해야한다면 선생님만 추가할 수 있음.
  - 로그인 기능은 따로 인증을 거치지 않는 방식으로 함.
    - 각 아바타마다 닉네임과 비밀번호를 임의로 지정한 후에 처음 접속하여 비밀번호를 설정하도록 한다.
    - 초기 학생 5명 아바타의 이름은 "학생1, 학생2, 학생3, 학생4, 학생5"로 함(가명).
    - 초기 선생 1명 아바타의 이름은 "soobing"으로 함.
    - 초기 게스트 1명 아바타의 이름은 "게스트"로 함. 게스트는 비밀번호 설정하지 않아도 되게 함. 비밀번호가 없이 들어올 수 있음.
- 맵 구성
  - 강원도 영월이라는 산골마을 컨셉임. 산과 강이 흐르는 느낌으로 작은 맵 필요
  - 학교 건물 필요
    - 학교 건물에 아바타가 입장시, 교실이 보이고 교실에는 칠판과 책상의자 그리고 뒷쪽에는 아이들의 작품을 걸어놓는 형식으로 구성
    - 칠판에는 강의자료를 붙여놓을 수 있도록 하고 그 강의자료를 클릭할 경우 html ppt로 제작된 화면이 크게 보일 수 있도록 함.
## 필요한 기능
- 아바타 비밀번호 변경 기능 (초기 비밀번호는 1234로 세팅, 숫자로만 비밀번호 설정 가능하도록 함)
- admin(선생님)에서 아바타 추가 가능 기능
- 선생, 학생 아바타는 기본적으로 핸드폰 소지
  - 선생님이 학생들 전체 또는 개인에게 문자, 투표, 설문을 진행할 수 있도록 함.
  - 선생님이 예를들면 설문을 만들어 전송한다면, 학생들이 이후에 접속시 핸드폰 알람이 울리고 해당 내용을 확인하거나 답할 수 있도록 함. 계속해서 확인하지 않으면 주기적으로 알람이 뜨도록 함.
